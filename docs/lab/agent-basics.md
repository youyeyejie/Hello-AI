# Lab 4: Agent 拆解

## 背景

你知道 Agent 会循环了，但还没自己拆过任务。

这一题要训练的是：把一个任务拆成可执行的步骤。

## 目标

把“帮我整理这份资料”拆成一个简单 Agent 流程。

## 输入材料

任务描述：

```text
帮我整理三篇关于 RAG 的笔记，提炼共同点和差异点。
```

## 提示与约束

你需要列出：

- 任务分解
- 需要哪些工具
- 什么时候停止
- 什么时候人工介入

## 参考答案

```text
1. 读取三篇笔记
2. 提取每篇笔记的核心观点
3. 对比共同点和差异点
4. 输出结构化总结
5. 如果某篇笔记太短或信息不足，人工介入补充
```

## 扩展思考

- 哪些任务适合 Agent
- 哪些任务不适合 Agent
- 为什么 Agent 需要停止条件

## 本地复现方式

1. 拿一组自己的笔记
2. 先手动拆任务
3. 再把拆法和模型输出对比
