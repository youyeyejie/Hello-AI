# Hello-AI

Hello-AI 是一条给中文学习者准备的 AI / LLM 入门路径。

它不追求内容堆满，而是把你真正需要的东西排成顺序。

## 先从这里进

- 新手先看 [新手起步](preface/start.md)
- 想看路线先看 [学习路线](preface/roadmap.md)
- 想直接进入基础先看 [AI 基础](basics/index.md)

## 这站主要解决什么

1. 你先知道 AI、LLM、Prompt、RAG、Agent 是什么
2. 你再学会怎么用
3. 你然后做出第一个小项目
4. 你最后开始理解评测、安全和部署

## 当前主线

- AI 基础
- Prompt 基础
- 工具使用
- RAG 入门
- Agent 入门
- Build 落地
- Eval 和 Safety

## 你会得到什么

- 一条清晰的学习顺序
- 一组能直接复用的练习
- 一套适合继续扩展的站点结构
