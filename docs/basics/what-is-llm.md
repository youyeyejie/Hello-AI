# 什么是 LLM

LLM 是 Large Language Model 的缩写，中文一般叫“大语言模型”。

它不是“会说话的机器人”这么简单，而是一个在大量文本上训练出来、能理解和生成语言的模型。

## 它擅长什么

- 回答问题
- 总结内容
- 改写文本
- 生成草稿
- 进行上下文内推断

## 它不擅长什么

- 保证绝对正确
- 自动知道最新事实
- 稳定处理长链条精确逻辑
- 无成本地记住所有上下文

## 先认识三个词

**Token**  
模型不是按“字”或“词”天然理解文本，而是按 Token 处理。Token 可以理解成模型的计量单位。

**Context Window**  
一次输入给模型的上下文上限。超过上限，前面的内容就可能看不见。

**Temperature**  
控制输出随机性的参数。值越高，输出通常越发散；值越低，通常越稳。

## 一个直观例子

如果你问：

```text
请用三句话解释 RAG，并给一个生活中的例子。
```

LLM 很适合把这类问题组织成自然语言答案。

但如果你问：

```text
请精确计算这份长文档里所有表格字段的差异，并保证零错误。
```

它就不一定是最可靠的工具，通常需要规则、检索、校验或人工复核一起配合。

## 学习建议

理解 LLM 时，先看这三个层面：

1. 它能做什么
2. 它不能做什么
3. 它和你的任务之间怎么配合
